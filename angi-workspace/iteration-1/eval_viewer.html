Key Findings

✓ What the Skill Fixed

  • Correct imports (@angi-ai/angi/client vs /server)
  • AngiAgent + processRequestStream (not createAngiHandler)
  • AngiNextProvider in layout (not AngiProvider)
  • useAngiComponent pattern (not useAngi register/unregister)
  • getState: () => {...} (not state: {...})
  • schema + execute (not parameters + handler)
  • execute returns void (not passed to Claude)
  • <Angi> boundary in parent, not inside component
